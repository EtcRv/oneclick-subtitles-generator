API Keys

Enter your API keys to use the application. These keys are stored locally in your browser and never sent to our servers.

Get Gemini API Key:

  1. Login to Google AI Studio
  2. Click 'Get API Key'
  3. Create a new key or select existing
  4. Copy your API key
  5. Paste it into the field above

Get YouTube API Key:

  1. Go to Google Cloud Console
  2. Create or select a project
  3. Enable 'YouTube Data API v3'
  4. Go to credentials
  5. Generate API key
